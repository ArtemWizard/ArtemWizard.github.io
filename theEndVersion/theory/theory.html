Квадрат суммы

Формула: (a+b)2 = a2 + 2ab +b2

Доказательство: (a+b)2 = (a+b)(a+b) = a2 + ab + ba +b2 = a2 + 2ab +b2

Буквенная расшифровка: Квадрат суммы двух чисел равен квадрату первого числа плюс удвоенное произведение первого числа на второе число плюс квадрат второго числа.

Примеры:

(5+2)2 = (5+2)(5+2) = 52 + 5*2 + 5*2 +22 = 52 + 2*10 +22 = 25 + 20 + 4 = 49

(4+3)2 = 42 + 2*4*3 +32 = 16 + 24 + 9 = 49

(9+2)2 = 92 + 2*9*2 +22 = 121

Квадрат разности

Формула: (a-b)2 = a2 - 2ab +b2

Доказательство: (a-b)2 = (a-b)(a-b) = a2 - ab - ba +b2 = a2 - 2ab +b2

Буквенная расшифровка: Квадрат разности двух чисел равен квадрату первого числа минус удвоенное произведение первого числа на второе число плюс квадрат второго числа.

Примеры:

(5-2)2 = (5-2)(5-2) = 52 - 5*2 - 5*2 + 22 = 52 - 2*10 + 22 = 25 - 20 + 4 = 9

(9-5)2 = 92 - 9*5*2 + 52 = 81 - 90 + 25 = 16

(9-2)2 = 92 - 2*9*2 + 22 = 49

Разность квадратов

Формула: a2 - b2 = (a-b)(a+b)

Доказательство: (a-b)(a+b) = a2 + ab - ab - b2 = a2 - b2

Буквенная расшифровка: Разность квадратов двух чисел равна произведению суммы этих чисел на их разность.

Примеры:

122 - 22 = (12-2)(12+2) = 10*14 = 140

92 - 72 = (9-7)(9+7) = 2*16 = 32

Разность кубов

Формула: a3 - b3 = (a-b)(a2 + ab + b2)

Доказательство: (a-b)(a2 + ab + b2) = a3 + a2b + ab2 - ba2 -ab2 - b3 = a3 - b3

Буквенная расшифровка: Разность кубов двух чисел равна произведению их разности и неполного квадрата суммы.

Примеры:

43 - 33 = (4-3)(42 + 3*4 + 32) = 1*(16 +12 + 9) = 37

93 - 53 = (9-5)(92 + 9*5 + 52) = 4*(81+45+25) = 4*151 = 604

Сумма кубов

Формула: a3 + b3 = (a+b)(a2 - ab + b2)

Доказательство: (a+b)(a2 - ab + b2) = a3 - a2b + ab2 + ba2 - ab2 + b3 = a3 + b3

Буквенная расшифровка: Сумма кубов двух чисел равна произведению их суммы и неполного квадрата разности.

Примеры:

43 + 33 = (4+3)(42 - 3*4 + 32) = 7*(16 - 12 + 9) = 91

33 + 73 = (3+7)(32 - 3*7 + 72) = 10*(9 - 21 + 49) = 370

Куб суммы

Куб разности

Формула: (a-b)3 = a3 - 3a2b + 3ab2 - b3

Доказательство: (a-b)3 = (a-b)(a-b)(a-b) = (a2 - ab - ba +b2)(a-b) = a3 - a2b - a2b + ab2 - ba2 + b2a + b2a - b3 = a3 - 3a2b + 3ab2 - b3

Буквенная расшифровка: Куб разности двух чисел равен кубу первого минус утроенное произведение квадрата первого на второе плюс утроенное произведение первого на квадрат второго минус куб второго.
